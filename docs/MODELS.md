# 模型选型指南

> 🧠 12 个国产大模型，怎么选？看完这篇你就懂了。

## 按用途分组

### 🆓 免费组（零成本起步）

这三个模型都有免费额度，注册就能用，不用花一分钱。

| 模型 | 免费额度 | 适合场景 |
|------|----------|----------|
| **GLM-4-Flash** | 永久免费 | 日常编程、学习、轻量任务 |
| **MiMo** (mimo-v2-flash) | 免费 | 大代码库分析（1M 上下文） |
| **DeepSeek-V3** | 新用户赠送额度 | 通用编程、性价比最高 |

### 💰 低成本组（日常编程好搭档）

这些模型按 token 计费，但价格极低，一天用下来可能就几毛钱。

| 模型 | 环境变量 | 适合场景 |
|------|----------|----------|
| **DeepSeek-V3** | `DEEPSEEK_API_KEY` | 代码生成、Bug 修复、重构 |
| **豆包** | `DOUBAO_API_KEY` | 快速响应场景、实时对话 |
| **Kimi** | `KIMI_API_KEY` | 超长上下文（128K）、大项目分析 |
| **天工 AI** | `TIANGONG_API_KEY` | 中文理解、文档生成 |
| **百川** | `BAICHUAN_API_KEY` | 中文处理、对话任务 |

### 🚀 复杂任务组（质量优先）

需要深度推理、复杂架构设计时上这些模型，成本稍高但输出质量更好。

| 模型 | 环境变量 | 适合场景 |
|------|----------|----------|
| **DeepSeek-R1** | `DEEPSEEK_API_KEY` | 复杂推理、算法设计、代码审查 |
| **通义千问** | `TONGYI_API_KEY` | 多模型切换、阿里生态集成 |
| **MiniMax** | `MINIMAX_API_KEY` | 中文长文本生成 |
| **文心一言** | `WENXIN_API_KEY` | 百度生态、企业场景 |
| **混元** | `HUNYUAN_API_KEY` | 腾讯生态、企业级应用 |

> ⚠️ 讯飞星火需要三个凭证（API Key + App ID + Secret Key），配置较复杂，推荐有经验的用户使用。

## GLM-4-Flash 的局限性说明

GLM-4-Flash 是我们最推荐的入门模型，因为它**永久免费**，但这不代表它没有局限：

| 方面 | 表现 | 说明 |
|------|------|------|
| 简单代码生成 | ✅ 很好 | 写工具脚本、CRUD 接口完全够用 |
| 中等复杂度任务 | ✅ 可用 | 重构、Bug 修复、测试生成都能做 |
| 复杂推理 | ⚠️ 偏弱 | 多步推理、算法设计不如 DeepSeek-R1 |
| 高峰期响应 | ⚠️ 偶有排队 | 免费模型，高峰期可能需要等几秒 |
| 超长上下文 | ⚠️ 128K | 不如 MiMo 的 1M，但对大多数项目够用 |

**总结**：GLM-4-Flash 是最好的入门选择。等你发现它跟不上你的需求时，再升级到 DeepSeek-V3/R1 也很方便——只需多配一个 API Key。

## 实用建议

### 🎓 新手/学生

**推荐组合**：GLM-4-Flash（主力） + MiMo（大文件分析）

```bash
export GLM_API_KEY="你的Key"     # 主力，免费
export MIMOAPIKEY="你的Key"       # 免费，1M 上下文
```

理由：零成本，功能齐全，够用。

### 💼 日常开发

**推荐组合**：DeepSeek-V3（主力） + GLM-4-Flash（备用）

```bash
export DEEPSEEK_API_KEY="sk-xxx"  # 主力，代码能力最强
export GLM_API_KEY="你的Key"       # 备用，免费兜底
```

理由：DeepSeek 代码能力国内顶级，GLM 免费兜底不浪费。

### 🏗️ 复杂项目

**推荐组合**：DeepSeek-R1（推理） + DeepSeek-V3（执行） + Kimi（大上下文）

```bash
export DEEPSEEK_API_KEY="sk-xxx"  # R1 + V3 共用
export KIMI_API_KEY="你的Key"      # 大项目分析
```

理由：R1 做架构决策和代码审查，V3 做日常代码生成，Kimi 处理大文件。

### 📚 学习 vs 💼 工作

| 场景 | 推荐模型 | 月成本 |
|------|----------|--------|
| 学习 Python | GLM-4-Flash | ¥0 |
| 刷 LeetCode | DeepSeek-V3 | ¥1-5 |
| 个人项目 | DeepSeek-V3 + GLM-4-Flash | ¥5-20 |
| 公司项目 | DeepSeek-R1 + V3 + Kimi | ¥30-100 |

> 💡 Oh My Coder 的智能路由会自动根据任务复杂度选择合适模型层级（LOW/MEDIUM/HIGH），你只需要配好 Key，剩下的交给系统。
