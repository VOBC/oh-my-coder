# 截图目录

本目录用于存放项目截图。

## 截图说明

| 文件名 | 说明 |
|--------|------|
| `web-ui.png` | Web 界面运行效果截图 |
| `cli-demo.png` | CLI 运行效果截图 |

## 添加截图

1. 运行 `python -m src.web.app` 启动 Web 界面，截图保存为 `web-ui.png`
2. 运行 CLI 命令，截图保存为 `cli-demo.png`
3. 推荐分辨率：1920×1080 或更高
4. 推荐格式：PNG，无压缩或低压缩

## 注意事项

- 请勿在截图中包含真实的 API Key 或敏感信息
- 建议使用深色模式截图（Web 界面支持 🌙 深色模式）
- 截图应清晰展示核心功能
