# 主动学习模块 (Self-Improving Agent)

## 概述

SelfImprovingAgent 让 Oh My Coder 在使用过程中自动学习和进化，持续提升代码质量。

## 核心能力

### 1. 执行反馈收集

每次任务完成后自动收集：
- Agent 执行结果（成功/失败）
- 修复次数和模式
- 用户干预记录

### 2. 错误模式分析

自动分析常见错误模式：
- 哪些 Agent 容易出错
- 哪些模型在什么场景下表现不佳
- 典型的修复路径

### 3. 策略自动优化

基于历史数据优化执行策略：
- 模型选择建议
- Agent 编排顺序优化
- 参数调优

### 4. 经验自动沉淀为 Skill

当任务满足以下任一条件时，自动将经验沉淀为 Skill 文件：
- 工具调用 ≥ 5 次
- 经历错误→修复流程
- 用户手动纠正
- 多步骤复杂任务

生成的 Skill 会自动：
- 分类存储（debugging / corrections / workflow / best-practices）
- 注入到后续任务的上下文中
- 通过 Tier 0 记忆系统按需加载

## 与其他模块的集成

- **分层记忆** — 学习结果写入 MemoryManager，通过 Tier 0/1 注入
- **Skill 系统** — 自动生成/更新 Skill 文件
- **Orchestrator** — 工作流完成后自动触发学习流程
