# Oh My Coder (OMC 中文版)

> 多智能体 AI 编程助手，支持国内大模型

原项目: https://github.com/Yeachan-Heo/oh-my-claudecode

## 目标
将 OMC 从 TypeScript + Claude Code 生态，重写为 Python + 多模型支持（DeepSeek/文心/通义/GLM）

## 核心特性
- 32 个专业智能体（架构师、UI设计师、安全专家等）
- 7 种执行模式
- 智能模型路由（节省 30-50% token）
- 中文优先，本土化适配

## 技术栈
- Python 3.10+
- FastAPI
- 支持模型: DeepSeek, 文心一言, 通义千问, ChatGLM
